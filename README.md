# wexample-wex-addon-dev-python

Python dev addon for wex

Version: 0.0.46

## Requirements

- Python >=3.10

## Dependencies

- attrs>=23.1.0
- cattrs>=23.1.0
- pylint
- pyright
- wexample-filestate-python==0.0.42
- wexample-wex-addon-app==0.0.44
- wexample-wex-core==6.0.46

## Installation

```bash
pip install wexample-wex-addon-dev-python
```
